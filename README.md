# Tumbes de Manglar de Carbono Azul
Este repositorio alberga el código, los datos y los modelos dedicados a la estimación, monitoreo y análisis del Carbono Azul en el ecosistema de manglares de Tumbes, Perú. Los manglares de Tumbes son uno de los sumideros de carbono más eficientes del planeta. El objetivo de este proyecto es utilizar herramientas tecnológicas (como teledetección, machine learning y procesamiento de datos geoespaciales) para cuantificar este almacenamiento y apoyar los esfuerzos de conservación y mitigación del cambio climático.
